## О UDWA Ai Education Platform

UDWA Ai Education Platform это платформа для обучения студентов разным наукам таким как :

- Информатика.
- Математика.
- Физика.
- География.
- История.
- Астрономия.
- Программирование.
- и другие.

Данная платформа призвнана для расширения знаний в области стандартных и профильных предметов, для повышения общегосударственного уровня подготовки и для повышения общего уровня знаний населения (в данный момент платформа находится на стадии реализации первого уровня, альфа версия).

## Основной стек технологий использованный в разработке

- Laravel 
- PHP
- MySQL
- Bootstrap
- JS
- HTML
- CSS
